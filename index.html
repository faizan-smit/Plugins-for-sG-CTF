SilentGamerZ Capture the Flag Server Rules

1. Camping / Skywalking / Skywalling

Camping for more than 30 seconds is forbidden, except when holding a stolen flag or healing on your own flag.

2. CTF SHOP

Bugging a CTF SHOP item is punishable, as well as generating glitches by combining two or more items.

3. Ghosting / LANing

Any form of ghosting / LANing is a punishable offense (Warning, Kick, Ban).

4. Bugs / Glitching

Using server/map bugs for advantages is strictly prohibited and punishable.

5. Teaming

Teaming with opponents for advantages or personal points is strictly prohibited.

6. Use of Special Knife/Weapon

Using special knife/weapon on your own teammate is prohibited.

7. Sabotaging

Sabotaging your own team by returning the enemy flag is strictly prohibited.

8. Bug Flag

Bugging the flag or leaving it in inaccessible zones is prohibited.

9. Restricted Zones

Entering inaccessible zones using CTF SHOP items or special knives is strictly prohibited.

10. Music/Sounds

Playing music/sounds on the server or repeating radio messages is forbidden.

11. AFK

Being AFK in the server or in Spectator mode will result in a kick when the server is full.

The server rules are an extension of the general rules, both must be fulfilled.

Visit SilentGamerZ.Org for more info!
